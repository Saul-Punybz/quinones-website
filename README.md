# Quinones Handyman & Roofing Services

Sitio web profesional para Quinones Handyman & Roofing Services, basado en Vega Baja, Puerto Rico.

## Tech Stack

- **Astro** — Static site framework
- **Tailwind CSS v4** — Utility-first styling
- **Netlify** — Deployment target
- **Vanilla JS** — No framework dependencies

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deploy to Netlify

1. Push this repo to GitHub
2. Connect the repo in [Netlify](https://app.netlify.com)
3. Netlify will auto-detect the build settings from `netlify.toml`
4. Deploy!

Or use the Netlify CLI:

```bash
npm i -g netlify-cli
netlify deploy --prod
```
